# Python Dice Roller

This repository contains a Python program that simulates rolling a dice.

## How It Works
- The program prompts the user to press Enter to roll the dice
- A random number between 1 and 6 is generated
- The user can continue rolling or quit the game

## Concepts Used
- while loop
- Conditional statements
- random module
- User input handling

## Purpose
To practice using loops, conditional logic, and the random module in Python through a simple interactive program.
